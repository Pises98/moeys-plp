# MoEYS PLP Frontend Clone | គំរូទម្រង់គេហទំព័រ PLP ក្រសួងអប់រំ

A beginner-friendly visual replica (frontend clone) of the Ministry of Education, Youth and Sport (MoEYS) Professional Learning Platform (PLP) website originally hosted at https://plp.moeys.gov.kh. This project is built as an educational training exercise to understand web layouts, HTML structure, and CSS styling.

គម្រោងនេះគឺជាការបង្កើតឡើងវិញនូវរូបរាងខាងក្រៅ (Frontend Clone) នៃគេហទំព័រវគ្គសិក្សាជំនាញវិជ្ជាជីវៈ (PLP) របស់ក្រសួងអប់រំ យុវជន និងកីឡា។ គម្រោងនេះធ្វើឡើងក្នុងគោលបំណងសិក្សាស្វែងយល់អំពីការរៀបចំរចនាសម្ព័ន្ធ HTML និងការដេគ័រម៉ូតដោយប្រើប្រាស់ CSS សម្រាប់អ្នកទើបតែចាប់ផ្តើមដំបូង។

---

## 🌟 Features / លក្ខណៈពិសេស
* **Official Theme Styling:** Utilizes the distinctive deep blue and warm gold color palettes typical of official Cambodian school portals.
* **Structured Layout:** Includes a complete functional header, dynamic hero announcement banner, course grid section, and detailed footer.
* **Fully Responsive:** Adapts smoothly across various computer screen sizes.

* **ពណ៌តំណាងផ្លូវការ៖** ប្រើប្រាស់ពណ៌ខៀវដិត និងពណ៌មាស ដែលជាពណ៌តំណាងឱ្យស្ថាប័នអប់រំជាតិ។
* **រចនាសម្ព័ន្ធច្បាស់លាស់៖** រួមមានផ្នែកក្បាលទំព័រ (Header) ផ្ទាំងបដាស្វាគមន៍ (Hero Banner) ប្រអប់បង្ហាញវគ្គសិក្សា (Course Grid) និងផ្នែកបាតទំព័រ (Footer)។
* **រៀបចំរាប់រយ៖** រៀបចំឡើងដោយមានភាពងាយស្រួលមើល មិនញីញៃ សម្រាប់ការសិក្សាគ្រឹះដំបូង។

---

## 🛠️ Built With / បង្កើតឡើងដោយប្រើប្រាស់
* **HTML5:** For the core skeleton and document structure (សម្រាប់គ្រោងឆ្អឹងវេបសាយ)
* **CSS3:** For the colors, typography, spacing, and styling (សម្រាប់លម្អពណ៌ និងអក្សរ)

---

## 🚀 How to Live Preview / របៀបបើកមើលគេហទំព័រពិត
You can view the live deployment of this project directly on the web via GitHub Pages!
1. Go to the repository **Settings**.
2. Click on **Pages** in the left sidebar.
3. Your live URL link will be displayed at the top of the section.

អ្នកអាចបើកមើលវេបសាយនេះដំណើរការផ្ទាល់នៅលើអ៊ីនធឺណិតតាមរយៈ GitHub Pages៖
1. ចូលទៅកាន់ផ្នែក **Settings** នៃ Repository នេះ។
2. ចុចលើពាក្យ **Pages** នៅរបារខាងឆ្វេង។
3. តំណភ្ជាប់ (Link) ផ្លូវការនឹងបង្ហាញនៅផ្នែកខាងលើនៃទំព័រនោះ។

## 📝 Project Context / បរិបទគម្រោង
This repository serves as a practical blueprint sample for teacher educators and junior web trainees under technical consulting guidelines for clean, semantic HTML documentation.

គម្រោងនេះដើរតួជាគំរូគ្រឹះសម្រាប់គ្រូបណ្តុះបណ្តាល និងសិស្សានុសិស្សក្នុងការអនុវត្តសរសេរកូដវេបសាយឱ្យមានរបៀបរៀបរយតាមបទដ្ឋានបច្ចេកទេស។
